7. Find the average review_scores_rating per neighborhood, and only show the ones above a 95, sorted in descending order of rating.<br>

The following aggregate function is used where we first group the documents according to the "neighbourhood_group_cleansed" field and then use the $avg method to calculate the average review_scores_rating for each of these groups. Then, the $match method filters out the results where the average is not greater than 95. Finally, the $sort method uses the avgReviewRating and the flag of -1 leads to a sorting in descending order. The following command, therefore, produces the desired output. <br>

```js
db.listings.aggregate([ {$group:{_id:"$neighbourhood_group_cleansed",avgReviewRating:{$avg:"$review_scores_rating"}}}, {$match: {avgReviewRating:{$gt:95}}}, {$sort: {avgReviewRating: -1}} ]);
```
